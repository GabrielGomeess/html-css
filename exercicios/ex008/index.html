Principais Formatações

Negrito ou Destaque

Nesta frase temos um termo em negrito usando a tag <b>, porém essa forma NÃO É SEMÂNTICA.

Já esta frase temos um termo em destaque usando a tag <strong>, e esta forma É SEMÂNTICA.

Itálico ou Ênfase

Nesta frase temos um termo em itálico usando a tag <i>, porém essa forma NÃO É SEMÂNTICA.

Nesta frase temos um termo em ênfase usando a tag <em>, e esta É SEMÂNTICA

Texto Marcado

Podemos também marcar um trecho do texto, como o marca texto tradicional amarelo, verde ou laranja, que você geralmente usa, utilizando a tag <mark>

Testando aqui se o seleto mark indicado no head, está funcionado ou não, para a marcação de um trecho do texto

Texto Pequeno e Texto Grande

Estamos criando um texto pequeno e um texto grande para formatações.

Texo Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado

Texo Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Exite também a tag <u>, para sublinhar um texto, porém ela NÃO É SEMÂNTICA
